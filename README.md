# Anti-Spinner

Демонстрация использования Device Orientation API для компенсации вращения элемента при повороте устройства.

## Описание

Этот проект показывает, как использовать Device Orientation API для создания элемента, который остается горизонтальным относительно горизонта при повороте устройства. Синий квадрат в центре экрана будет автоматически компенсировать поворот телефона вокруг оси Z, оставаясь в горизонтальном положении.

## Используемые технологии

- HTML5
- CSS3 (3D трансформации)
- JavaScript (Device Orientation API)
- Web App Manifest

## Файлы проекта

- `index.html` - основная HTML-страница
- `style.css` - стили для страницы и квадрата
- `script.js` - JavaScript код для обработки ориентации устройства
- `manifest.json` - веб-манифест для PWA

## Как это работает

1. При открытии страницы на мобильном устройстве может потребоваться разрешение на доступ к данным ориентации устройства.
2. После получения разрешения страница начинает отслеживать изменение ориентации устройства.
3. Синий квадрат автоматически поворачивается в обратном направлении относительно поворота устройства вокруг оси Z, оставаясь горизонтальным.

## Поддерживаемые API

- `deviceorientation` - основное событие для отслеживания ориентации устройства
- `deviceorientationabsolute` - более точное событие (поддерживается не всеми браузерами)
- `devicemotion` - резервное событие для получения данных об ускорении

## Совместимость

Проект работает на большинстве современных мобильных браузеров, поддерживающих Device Orientation API:
- Safari (iOS 13+)
- Chrome (Android/iOS)
- Firefox (Android)

## Особенности реализации

- Используется CSS3 3D трансформации для плавного поворота элемента
- Поддержка запроса разрешения для iOS 13+ (требуется пользовательское взаимодействие)
- Адаптивный дизайн для различных размеров экранов
- Градиентное оформление квадрата с тенями для лучшего визуального восприятия
- Поддержка Web App Manifest для установки как PWA
- Оптимизирована производительность с использованием `will-change` и оптимизированных переходов

## Известные ограничения

- На некоторых устройствах или в браузерах данные ориентации могут быть недоступны
- В браузерах для настольных компьютеров данные ориентации обычно отсутствуют
- Требуется HTTPS для работы Device Orientation API в большинстве современных браузеров
- На iOS 13+ требуется явное пользовательское взаимодействие для получения разрешения на доступ к данным ориентации

## Установка

1. Клонируйте репозиторий:
   ```
   git clone https://github.com/vchemonin/anti-spinner.git
   ```

2. Откройте `index.html` в браузере или разместите файлы на веб-сервере с поддержкой HTTPS.

## Использование

1. Откройте страницу на мобильном устройстве.
2. При появлении кнопки "Разрешить доступ к ориентации устройства" нажмите на неё.
3. Поверните устройство вокруг оси Z (поворот плоскости экрана) и наблюдайте, как квадрат остается горизонтальным.
